# Anuj Portfolio Frontend

Fronted Repo for Anuj's Portfolio

Stack
- Next.js
- Tailwind
- Shadcn UI
- Typescript

To run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
